# Installation Guide

Follow these steps to install and run crusadeOS on your local machine:

## Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher recommended)

## 1. Clone the Repository
```
git clone https://github.com/notpran/crusadeOS
cd crusadeOS
```


## 4. Start crusadeOS
```
npm install
npm start
```

## 6. Access crusadeOS
crusadeOS will automatically open on startup, if it dossent
open your browser and go to:
```
http://localhost:8000
```
